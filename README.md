# Golf-data-analysis
This project performs descriptive data analysis on two golf-related datasets.

Datasets Used
-------------
1. Golf Swing Data
2. Golf Play Extended

Objectives
----------
• Explore the datasets using pandas
• Identify missing values
• Generate summary statistics
• Create visualizations
• Answer descriptive analysis questions

Repository Structure
--------------------
data/       -> raw datasets
notebooks/  -> Jupyter/Colab notebooks
images/     -> visualization outputs

Tools Used
----------
Python
Pandas
Matplotlib
Seaborn
Google Colab
